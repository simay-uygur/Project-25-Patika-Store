# Project-25-Patika-Store
This is the product management system for a virtual store selling electronic goods.



The task is: 

```
The name of the virtual store will be "PatikaStore."
In the store, brands will be created, and products will be associated with these brands.

id: The unique number of the brand registered in the system.
name: The name of the brand.

Brands should always be listed alphabetically.

Brands should be added statically in the following order from code blocks:

Samsung, Lenovo, Apple, Huawei, Casper, Asus, HP, Xiaomi, Monster

Currently, the store plans to sell 2 types of product groups: Mobile Phones and Notebooks.

Later on, different product groups should be able to be added.

Mobile Phone Product Characteristics:

Unique number of the product registered in the system
Unit price
Discount rate
Stock quantity
Product name
Brand information (Brands available in the system will be used)
Phone memory information (128 GB, 64 GB)
Screen Size (6.1 inches)
Battery Capacity (4000 mAh)
RAM (6 GB)
Color (Black, Red, Blue)
Notebook Product Characteristics:

Unique number of the product registered in the system
Unit price
Discount rate
Stock quantity
Product name
Brand information (Brands available in the system will be used)
RAM (8 GB)
Storage (512 SSD)
Screen Size (14 inches)
Users should be able to list products in the relevant category (Notebooks, Mobile Phones, etc.) through the system.

Products should be displayed in a tabular format on the console screen (System.out.format() can be used).

Users should be able to add products and select the product group (Mobile Phone, Notebook, etc.).

Users should be able to delete products based on their unique numbers.

Users should be able to filter and list products based on their unique numbers and brands.

```
